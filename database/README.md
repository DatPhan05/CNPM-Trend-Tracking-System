# Database Directory

This directory is structured for raw SQL-based relational database architecture, as specified in the project requirements.

## Structure
- `schema/`: Database schema definitions
- `migrations/`: SQL migration scripts
- `tables/`: Table creation scripts
- `views/`: View definitions
- `procedures/`: Stored procedures
- `functions/`: Database functions
- `constraints/`: Primary, foreign key, and check constraints
- `indexes/`: Index definitions
- `seeds/`: Initial data seed scripts
- `config/`: Database configurations
- `docs/`: Database documentation (ERD, data dictionary)
